\## ✨ Nombre del proyecto



\*\*Floralia\*\*

\*Una floristería online pensada para emocionar, regalar y sorprender.\*



\---



\## 📌 Descripción del proyecto



Este proyecto intermodular consiste en el desarrollo de una \*\*floristería online\*\* orientada a todo tipo de público, con una propuesta centrada en la venta de flores y arreglos florales de forma cómoda, visual y cercana.\[file:2]



La empresa nace con la idea de ofrecer \*\*flores bonitas, ecológicas y de buena calidad\*\* para cualquier persona que quiera tener un detalle especial, celebrar un momento importante o simplemente regalar algo con significado.\[file:2]



A través de este proyecto se busca representar una empresa ficticia realista y coherente, tal y como se plantea en el Proyecto Intermodular de 1º de DAW, donde la web debe responder a una necesidad concreta y creíble.\[file:2]



\---



\## 🎯 Objetivo de la empresa



La finalidad de la floristería es acercar al público una forma sencilla y agradable de comprar flores, facilitando tanto el regalo en ocasiones especiales como la compra personal por gusto o decoración.\[file:2]



La propuesta de valor de la empresa se basa en tres pilares principales:



\- \*\*Calidad\*\*, ofreciendo flores y composiciones florales cuidadas.

\- \*\*Estética\*\*, con productos bonitos y presentados con gusto.

\- \*\*Compromiso ecológico\*\*, apostando por una imagen más natural y responsable.



\---



\## 👥 Público objetivo



La floristería está dirigida a \*\*todo el público\*\*, ya que sus productos pueden adaptarse a muchas situaciones y necesidades distintas.\[file:2]



Entre los posibles clientes se encuentran:



\- Personas que quieren hacer un regalo por cumpleaños, aniversarios o celebraciones.

\- Usuarios que desean enviar flores a otra persona.

\- Clientes que buscan decorar espacios con flores naturales.

\- Personas que valoran productos visualmente atractivos y de buena calidad.



\---



\## 🌼 Qué problema resuelve



Muchas veces, encontrar un regalo bonito, rápido y con valor emocional no es tan sencillo como parece.\[file:2]



Esta floristería busca dar solución a esa necesidad ofreciendo un espacio online donde el usuario pueda encontrar flores y arreglos florales de calidad, elegir el producto que más le guste y enviarlo a un destinatario concreto de forma cómoda y clara.\[file:2]



De este modo, la empresa aporta una solución útil para quienes quieren \*\*regalar algo especial\*\*, \*\*tener un detalle inesperado\*\* o \*\*comprar flores para una ocasión importante sin complicaciones\*\*.



\---



\## 🛍️ Qué ofrecerá la web



La web que se desarrollará posteriormente permitirá a los usuarios consultar el catálogo de la floristería y visualizar distintos productos, como \*\*flores sueltas\*\* y \*\*otros arreglos florales\*\* preparados para diferentes ocasiones.\[file:2]



Además, la plataforma estará pensada para que el usuario pueda:



\- Ver los productos disponibles.

\- Consultar información básica de cada producto.

\- Comprar online de forma sencilla.

\- Elegir a qué destinatario se enviará el pedido.



Todo ello se planteará con una presentación visual cuidada, coherente con la imagen de una floristería elegante, cercana y profesional, algo que también encaja con lo que se pide en el módulo de Lenguajes de Marcas para una web corporativa creíble y bien presentada.\[file:2]



\---



\## 💡 Idea principal del proyecto



La idea principal es crear una web de floristería que combine una estética agradable con una funcionalidad útil para el usuario.\[file:2]



No se trata solo de mostrar productos, sino de construir la base de una empresa ficticia con sentido real dentro del mercado, donde la web actúe como escaparate digital y como punto de compra para clientes que buscan flores de calidad con una experiencia sencilla y atractiva.\[file:2]



\---



\## 🛠️ Tecnologías previstas



\### Frontend

\- \*\*HTML5\*\*

\- \*\*CSS3\*\*



\### Backend

\- \*\*Java 25 LTS\*\*

\- \*\*JDBC\*\*

\- \*\*Maven\*\*



\### Base de datos

\- \*\*SQL\*\*

\- \*\*MariaDB\*\*

\- \*\*DBeaver\*\*



Estas tecnologías encajan con los módulos del proyecto, ya que el portal web deberá construirse con HTML y CSS, mientras que la aplicación de programación debe gestionar datos reales mediante Java y JDBC sobre una base de datos coherente con la empresa ficticia.\[file:2]



\---



\## 🌷 Conclusión



Este proyecto tiene como base la creación de una floristería online accesible, atractiva y funcional, orientada a todo el público y enfocada en ofrecer flores bonitas, ecológicas y de calidad.\[file:2]



La futura web permitirá visualizar productos, comprarlos y enviarlos a un destinatario elegido, convirtiéndose en una propuesta realista y adecuada para desarrollar el Proyecto Intermodular de 1º de DAW.\[file:2]
